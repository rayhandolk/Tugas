Selamat Datang di Aplikasi Perpustakaan Ray

Aplikasi ini dirancang untuk membantu mengelola data peminjaman buku di perpustakaan.

Langkah Menggunakan Aplikasi:

  1. Buka halaman Transaksi untuk melihat daftar buku yang dipinjam atau menambahkan data peminjaman baru.
  2. Isi formulir peminjaman dengan data yang valid, lalu klik tombol "Simpan".
  3. Kunjungi halaman Profil untuk melihat deskripsi lengkap tentang aplikasi ini.
  4. Gunakan tombol Logout untuk keluar dari akun Admin Perpustakaan dan kembali ke menu Login
