Profil Lengkap Aplikasi

1. Deskripsi Aplikasi

Aplikasi Admin Perpustakaan ini adalah sebuah purwarupa (prototype) sistem informasi berbasis web yang dirancang untuk mengelola data transaksi peminjaman buku. Aplikasi ini dibuat sebagai tugas Ujian Akhir Semester (UAS) untuk mata kuliah Pemrograman Web 1 program studi Sistem Informasi, Fakultas Ilmu Komputer, Universitas Pamulang.

2. Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan beberapa teknologi web modern, yaitu:

  • HTML5: Digunakan untuk membangun struktur dan kerangka dasar semua halaman web.
  • CSS3: Digunakan untuk memberikan styling kustom seperti warna latar header/footer dan penyesuaian tata letak untuk mempercantik tampilan.
  • Bootstrap 5: Sebagai framework CSS utama untuk memastikan desain yang responsif, modern, dan konsisten di berbagai perangkat. Komponen seperti Navigasi, Card, Form, dan Modal berasal dari Bootstrap.
  • JavaScript: Digunakan untuk memberikan interaktivitas pada sisi klien, terutama untuk fitur validasi formulir di halaman transaksi agar tidak ada data kosong yang dikirim.

3. Panduan Penggunaan Halaman

Berikut adalah penjelasan fungsi dari setiap halaman yang ada di aplikasi ini:

Halaman Login

Merupakan halaman simulasi untuk proses otentikasi admin. Setelah mengisi username dan password lalu menekan tombol Login, sebuah pop-up akan muncul untuk menandakan login berhasil, kemudian pengguna akan diarahkan kembali ke Halaman Beranda.

Halaman Beranda

Merupakan halaman utama saat aplikasi dibuka. Halaman ini berisi ucapan selamat datang dan ringkasan langkah-langkah penggunaan aplikasi secara umum.

Halaman Transaksi

Halaman ini adalah inti dari aplikasi, tempat admin mengelola data peminjaman. Pengguna dapat:

  • Mengisi Form Transaksi: Memasukkan data peminjaman buku baru yang mencakup judul buku, nama peminjam, dan tanggal pinjam.
  • Validasi Form: Jika salah satu kolom pada form tidak diisi dan tombol "Simpan" ditekan, sistem akan menampilkan pesan kesalahan berkat validasi JavaScript.
  • Melihat Daftar Transaksi: Menampilkan data transaksi yang sudah ada dalam bentuk tabel agar mudah dibaca.
Halaman Profil

Halaman yang sedang Anda lihat sekarang. Berisi deskripsi lengkap mengenai aplikasi, teknologi yang dipakai, panduan penggunaan, dan informasi tentang pembuat aplikasi.

Tombol Logout

Berfungsi untuk mensimulasikan proses keluar atau mengakhiri sesi admin. Ketika tombol "Logout" di bagian kanan atas navbar ditekan, pengguna akan langsung dialihkan kembali ke Halaman Login, yang merupakan halaman awal sebelum masuk ke dalam sistem.
